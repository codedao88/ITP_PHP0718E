Giới thiệu CSS

CSS là đoạn mã để định dạng (giao diện) cho các thành phần HTML

Có 3 cách để nhúng CSS vào webpage

- Cách 1: Nhúng trực tiếp css vào thuộc tính Style của HTML

- Cách 2: Nhúng CSS trong cặp thẻ style và truy cập vào thành phần html sử dụng bộ chọn css

- Cách 3: Nhúng file css độc lập vào webpage sử dụng thẻ link
